# Sistema de Doceria SQL 🍰

Projeto acadêmico desenvolvido para praticar modelagem de banco de dados e consultas SQL através de um sistema de gerenciamento para uma doceria.

## Tecnologias

* SQL
* Banco de Dados Relacional

## Estrutura do Projeto

### doceria_ddl.sql

Contém os comandos de criação das tabelas e estrutura do banco de dados.

### doceria_dml.sql

Contém comandos para inserção e manipulação dos dados.

### doceria_queries.sql

Contém consultas SQL para extração e análise das informações cadastradas.

## Funcionalidades

* Cadastro de produtos
* Organização de informações da doceria
* Consultas e relatórios em SQL
* Manipulação de dados

## Objetivo

Aplicar conceitos de modelagem de dados, criação de tabelas, inserção de registros e consultas em banco de dados relacionais.
